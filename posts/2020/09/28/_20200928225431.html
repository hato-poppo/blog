# sshd_config の設定変更

以前の記事

で挙げていた項目の中でも出来るだけ早めにやっておきたいこと

特にパスワード認証は早めに無効化しておきたい

# 手順

/etc/ssh/sshd_configファイルの編集をする

$ sudo vi /etc/ssh/sshd_config

今回設定の変更を行うのは後述の2点

# パスワード認証の無効化

- PasswordAuthentication yes
+ PasswordAuthentication no

# rootユーザーのssh接続を無効化

- PermitRootLogin yes
+ PermitRootLogin no

# 目的

ssh接続の設定を適切に行い、不正アクセスからサーバーを守る
