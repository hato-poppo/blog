**特にパスワード認証は早めに無効化しておきたい**  

### 手順

`/etc/ssh/sshd_config`ファイルの編集をする
```sh
$ sudo vi /etc/ssh/sshd_config
```

今回設定の変更を行うのは後述の2点

#### パスワード認証の無効化

```sh
# 変更前
PasswordAuthentication yes

# 変更後
PasswordAuthentication no
```

#### rootユーザーのssh接続を無効化

```sh
# 変更前
PermitRootLogin yes

# 変更後
PermitRootLogin no
```
  
### 目的

ssh接続の設定を適切に行い、不正アクセスからサーバーを守る
